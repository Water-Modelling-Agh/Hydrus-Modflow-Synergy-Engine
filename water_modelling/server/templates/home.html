Create a new project

 To prepare Hydrus-Modflow simulation you need to create a new project and fill the form to define additional configuration. Later you will have an access to the existing project through "List of Projects".
To create a new project click "Create Project".

Create Project
Choose an existing project

 You can open an existing Hydrus-Modflow project to simulate it. "List of Projects" tab contains all projects. Additional actions defined next to the simulation will help manage it.
Click "Choose Project" for redirection to the list of projects.

Choose Project
Project Summary

 After choosing project for simulation you will be redirected to the Project Summary.

Configure simulation

 After choosing a project for simulation you can click "Simulate" tab to upload Modflow and Hydrus models and also to define subareas assigned to Hydrus models.

Upload Modflow model

 First step of configuring simulation is to upload correct Modflow model. You need to drag and drop .rar or .zip archive with model files placed directly in the root.

Upload Hydrus model

 Second step is to upload correct Hydrus models. You need to drag and drop .rar or .zip archive with model files placed directly in the root. You can upload more than one Hydrus model.

Define method

 Next step is to choose one method for defining shapes. There are two options: "Recharge RCH" and "Define shapes manually". Choose one which is better for your simulation.

Recharge RCH method

Manual method

Run Simulation

 After defining all shapes you will be redirected to the Simulation Summary. Click "Run" to start simulation. If it ends with success, the button to download results will appear at the bottom of the screen. You will be able also to download results from the List of Projects.
